Java (linguagem de programação)

Java é uma linguagem de programação orientada a objetos , de alto nível , de uso geral e com segurança de memória . Ela foi criada para permitir que os programadores escrevam uma vez, executem em qualquer lugar ( WORA ), [ 18 ] o que significa que o código Java compilado pode ser executado em todas as plataformas que suportam Java sem a necessidade de recompilação.

Princípios

Existiam cinco objetivos principais na criação da linguagem Java:

  1. Deve ser simples, orientado a objetos e familiar.
  2. Deve ser robusto e seguro.
  3. Deve ser neutro em termos de arquitetura e portátil.
  4. Deve ser executado com alto desempenho.
  5. Ele deve ser interpretado , encadeado e dinâmico .

Sistema de execução

Um objetivo de design do Java é a portabilidade , o que significa que programas escritos para a plataforma Java devem rodar de forma similar em qualquer combinação de hardware e sistema operacional com suporte de tempo de execução adequado. Isso é alcançado compilando o código da linguagem Java para uma representação intermediária chamada bytecode Java , em vez de diretamente para código de máquina específico da arquitetura . Instruções de bytecode Java são análogas ao código de máquina, mas elas são destinadas a serem executadas por uma máquina virtual (VM) escrita especificamente para o hardware host. Usuários finais geralmente usam um Java Runtime Environment (JRE) instalado em seus dispositivos para aplicativos Java autônomos ou um navegador web para applets Java .
