Java é uma linguagem de programação orientada a objetos , de alto nível , de uso geral e com segurança de memória . Ela foi criada para permitir que os programadores escrevam uma vez,
executem em qualquer lugar ( WORA ), [ 18 ] o que significa que o código Java compilado pode ser executado em todas as plataformas que suportam Java sem a necessidade de recompilação.
