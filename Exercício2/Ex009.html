Algoritmo
Conjunto de passos lógicos e bem definidos que descrevem como resolver um problema ou executar uma tarefa.
Compilador
Programa que traduz o código-fonte escrito em uma linguagem de programação para código de máquina (entendido pelo computador).
Banco de Dados
Estrutura organizada para armazenar, gerenciar e recuperar informações de forma eficiente.
API (Application Programming Interface)
Conjunto de regras e padrões que permitem que diferentes sistemas ou aplicações se comuniquem entre si.
Framework
Estrutura de suporte que fornece ferramentas, bibliotecas e boas práticas para acelerar o desenvolvimento de software.
