Java (linguagem de programação)

Wikipedia

Historia

James Gosling , Mike Sheridan e Patrick Naughton iniciaram o projeto da linguagem Java em junho de 1991. Java foi originalmente projetado para televisão interativa, mas era muito avançado para a indústria de televisão a cabo digital na época. A linguagem foi inicialmente chamada de Oak , em homenagem a um carvalho que ficava do lado de fora do escritório de Gosling. Mais tarde, o projeto passou a se chamar Green e finalmente foi renomeado para Java , de Java coffee , um tipo de café da Indonésia . Gosling projetou Java com uma sintaxe no estilo C / C++ que os programadores de sistemas e aplicativos achariam familiar.

Sintaxe

A sintaxe de Java é amplamente influenciada por C++ e C . Ao contrário de C++, que combina a sintaxe para programação estruturada, genérica e orientada a objetos, Java foi construída quase exclusivamente como uma linguagem orientada a objetos. Todo o código é escrito dentro de classes, e cada item de dados é um objeto, com exceção dos tipos de dados primitivos (ou seja, inteiros, números de ponto flutuante, valores booleanos e caracteres), que não são objetos por razões de desempenho. Java reutiliza alguns aspectos populares de C++ (como o printf método ).

implementações

A Oracle Corporation detém a implementação oficial da plataforma Java SE, devido à aquisição da Sun Microsystems em 27 de janeiro de 2010. Esta implementação é baseada na implementação original do Java pela Sun. A implementação da Oracle está disponível para Windows , macOS , Linux e Solaris . Como o Java não possui nenhuma padronização formal reconhecida pela Ecma International , ISO/IEC, ANSI ou outras organizações de padronização terceirizadas, a implementação da Oracle é o padrão de fato .
